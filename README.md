Project Title:
The Luxe Haven – Luxury Travel Agency Website

Description
This is a responsive travel agency website built using HTML, CSS, and JavaScript.
The website showcases tour packages, itineraries, traveler reviews, and includes an inquiry form.
It features a premium black–gold luxury theme with smooth animations, a dark/light mode switch, and a sticky call-to-action button.
This project was created as part of my Web Development assignment.
Key Features

✔ Responsive Design
Works smoothly on mobile, tablet, and desktop.

✔ Dark & Light Theme Toggle
User can switch between dark and light mode, stored in localStorage.

✔ Animated Gold Glow Effects
Buttons, dividers, and CTAs have animated gold highlights for a premium feel.

✔ Scroll Reveal Animations
Sections fade and slide in when scrolled into view using IntersectionObserver.

✔ Sticky Floating CTA Button
A gold animated button appears after scrolling to encourage contacting.

✔ Packages Section
Shows 3 tour packages with price, features, and “Book Now” options.

✔ Why Choose Us Section
Highlights agency strengths.

✔ Itinerary & Reviews
Well-structured itinerary cards and traveller feedback.

✔ Contact Form
Includes inputs for name, email, phone, package selection, dates, and custom message.


Technologies Used
HTML5
CSS3 (variables, gradients, shadows, animations)
JavaScript (Vanilla)
Google Fonts (Montserrat)

How to Run the Project
Download or copy the project folder.
Open the index.html file in any browser (Chrome recommended).
All animations, theme toggles, and sticky CTA buttons will work automatically.
